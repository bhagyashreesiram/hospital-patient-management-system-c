# hospital-patient-management-system-c
Hospital Patient Management System using Queue (Data Structures in C)
## Overview

This project is developed as a part of my Data Structures subject. It is a menu-driven program in C that manages hospital patients using the Queue (FIFO) concept. The system handles patient records across different departments and simulates how patients are served in order.

## Features

* Add new patient details
* Serve patients based on queue (FIFO)
* View all patients department-wise
* Search patient using ID
* Discharge patient from system
* Display overall statistics

## Concepts Used

* Circular Queue
* Structures in C
* Menu-driven programming

## Departments Included

* Orthopedic
* Dermatology
* Cardiology
* General

## How to Run the Program

1. Compile the code:
   gcc src/main.c -o hospital

2. Run the program:
   ./hospital
